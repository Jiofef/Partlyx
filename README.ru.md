## Что такое Partlyx?
Partlyx позволяет вам создавать схемы (рецепты) разной сложности, состоящие из компонентов, коими могут быть любые ранее созданные ресурсы.

-----

## Какую проблему решает Partlyx?

Есть множество систем, в которых ресурсы/сущности:
* Состоят из других ресурсов
* Образуют глубокие и разветвлённые графы зависимостей
* Требуют визуализации и подсчёта фундаментальных/базовых элементов

Например: Производственные цепочки, крафты в играх, экономические модели.

Выполнение таких задач с помощью стандартных калькуляторов и таблиц часто либо невозможно, либо малоэффективно, а так же не очень наглядно. Например, если нам понадобится изменить рецепт какого-то из частоиспользуемых элементов, нам может понадобиться пересчитывать его составляющие для всех элементов, которые его используют.

## Как Partlyx решает эту проблему?

* Он позволяет описывать сущности как граф зависимостей
* Визуализирует многоуровневую структуру
* Считает суммарную стоимость/вес/цену по всему графу
* Работает с абстрактными типами ресурсов

Приложение заточено не под конкретную ситуацию, а под класс задач

Приложение отличается гибкостью и ориентировано на наглядную визуализацию сложных структур

-----

## Технические особенности приложения:

Приложение написано на языке C# с использованием Avalonia - современного кроссплатформенного фреймворка

Структура MVVM, разделение на 6 слоёв: Core <- Infrastructure <- Services <- ViewModels <- UI (View), а так же Tests

Применённые технологии:
* Dependency Injection для гибкости и модульности
* SQLite для баз данных
* ReactiveUI для реактивности во View и ViewModels
